# Examples

Worked examples of DBPilot AI in action — natural-language questions, the
SQL they generate, and the result. Populated starting Phase 3 (Text-to-SQL)
once there's a working end-to-end query path.

Planned format per example: `examples/<NN>-<slug>/README.md` with the
question asked, generated SQL, explanation, and (once Phase 8 lands) the
resulting chart.
