# Deployment Configs

| File | Purpose |
|---|---|
| `docker-compose.yml` | Local full-stack development (Postgres + backend + frontend) |
| `vercel.json` | Frontend deployment config for Vercel |
| `railway.json` | Backend deployment config for Railway |
| `render.yaml` | Backend deployment config for Render (alternative to Railway) |

See [../docs/deployment.md](../docs/deployment.md) for the full deployment guide.
